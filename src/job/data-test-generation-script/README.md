# README #

Outils permettant la génération de fichier de RDO ServicePrestation
<br>
Cet outil n'est pas géré par l'IAC, le service est noté "enabled: false au niveau du values.yaml

Pour l'éxecuter :

- s'il n'apparait pas comme executable, lancer un "Reload All Gradle projects"
- Modifier les valeurs idDeclarant ou autres dans le code
- executer le via l'onglet "Services" en spécifiant le profil dev
- Récupérer le fichier dans le répertoire C:\tmp\_cuc
