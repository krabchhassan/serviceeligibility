### Description

Cet OMU permet d'extraire les personnes n'ayant pas de droits TP générés.

La liste des personnes n'ayant pas de droits TP est extraite dans un fichier csv sur l'espace personnel.

### Schéma des interactions
N/A

### Evènements publiés
2 user-events sont publiés :
    - un au lancement de l'OMU pour avertir l'utilisateur qu'il s'agit d'un traitement long
    - un à la fin de l'OMU pour indiquer que le traitement est terminé
