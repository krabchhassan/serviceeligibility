### Description

Cet OMU permet de recycler une liste de déclencheurs en anomalie.

La liste des déclencheurs traités est fournie via un fichier à déposer sur l'espace personnel.

Le fichier contient les id techniques des déclencheurs sous le format suivant : 
- un id par ligne
- pas de guillemets, ni de virgules
